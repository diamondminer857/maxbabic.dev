A cross-platform mobile app built with Ionic and plain JavaScript, wrapped
with Capacitor for native deployment. Integrates the Imagga API for
automatic photo classification, tagging, and face detection.

Built as one of the assignments for the TAMZ (mobile application
development) course at FEI.
